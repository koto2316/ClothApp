# ClothApp

Android Bootcampで制作したアプリです。

## 概要
Jetpack Compose を使って作成した服の一覧・詳細表示アプリです。

## 使用技術
- Kotlin
- Jetpack Compose
- Android Studio

## 機能
- 商品一覧画面/詳細画面/カゴの中身の画面がある
- カゴに追加/削除機能
- 商品をクリックすると詳細画面へ
- お気に入り追加機能
- カゴの中身一覧と合計金額が表示

## 学んだこと
- Composeの基本
- 開発環境の構築
- 画面遷移の仕方
- DataStoreでのデータ永続化について
- ファイル分割の方法
- Column/Row/Box/LazyColumn
- 画面遷移しても状態を保存する
